Lover's Puzzle (MyGift) 
Lover's Puzzle is a lightweight puzzle-solving game developed using the Qt framework. Originally created as a gift for my ex-girlfriend.

[![Watch the video on YouTube](https://img.youtube.com)](https://www.youtube.com/watch?v=Uh1RSEjXI1Q)

🛠 Tech Stack
Language: C++ / QML
Framework: Qt Framework
Platform: Android (via Qt for Android) & Desktop

📱 Download the App from Google Play: [Lover's Puzzle on Google Play Store]([https://github.com/wallrue/QT_Puzzle-Game/assets](https://play.google.com/store/apps/details?id=org.qtproject.example.MyGift))
